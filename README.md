<h2 align="center">🏃‍♂️Instruções</h1>

Clone o projeto e acesse a pasta.

```bash
$ git clone https://github.com/mujapira/Todo-Mobile.git
$ cd todo-mobile
```

Para iniciá-lo, siga os passos abaixo:
```bash
# Instalar as dependências
$ npm install
# Iniciar o projeto
$ npx expo start
```
